# Unit 5: Turing Machine

## 🎯 Learning Objectives
- Formal definition of TM
- Design TMs for various tasks
- TM variants (multi-tape, NTM)
- Church-Turing Thesis
- Recursive and RE languages

---

## 1.1 What is a TM?

**TM = (Q, Σ, Γ, δ, q₀, B, F)**

- Infinite tape
- Read/write head
- Finite control

---

## 1.2 TM Variants

- Multi-tape TM
- Nondeterministic TM
- Universal TM

All equally powerful!

---

## 1.3 Church-Turing Thesis

"Any computable function can be computed by a TM"

---

## 1.4 Recursive vs RE

- **Recursive:** TM always halts
- **RE:** TM may loop on non-members
