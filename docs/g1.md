# Assignment 5: CFG to CNF Conversion

## Problem Statement
Write a C program to convert Context Free Grammar (CFG) to Chomsky Normal Form (CNF).

## Theory

### Context Free Grammar (CFG)
A CFG is defined as G = (V, T, P, S) where:
- **V**: Set of non-terminal symbols (variables)
- **T**: Set of terminal symbols
- **P**: Set of production rules
- **S**: Start symbol

### Chomsky Normal Form (CNF)
A CFG is in CNF if all production rules are of the form:
1. **A → BC** (Two non-terminals)
2. **A → a** (Single terminal)
3. **S → ε** (If ε is in the language, only start symbol can produce it)

### Why CNF?
- Simplifies parsing algorithms (CYK algorithm)
- Eliminates ambiguity in productions
- Standard form for theoretical analysis
- Easier to implement in compilers

### Conversion Steps

1. **Add New Start Symbol** (if needed)
2. **Remove Null Productions** (A → ε)
3. **Remove Unit Productions** (A → B)
4. **Convert to CNF**:
   - Replace terminals in mixed productions with new non-terminals
   - Break long productions into binary productions

## Example

### Original CFG
```
S → aSb | ε
```

### Step 1: Remove ε-production
```
S → aSb | ab
```

### Step 2: Convert to CNF
Introduce new non-terminals:
```
S → AB | AC
A → a
B → SC
C → b
```

## Code

```c
#include <stdio.h>
#include <string.h>

struct Production {
    char lhs;
    char rhs[10][10];
    int count;
};

int main() {
    struct Production prods[10];
    int n, i, j, k;
    char newVar = 'A';
    
    printf("Enter number of productions: ");
    scanf("%d", &n);
    
    printf("Enter productions (format: A->aB|b):\n");
    for(i = 0; i < n; i++) {
        char input[50];
        scanf("%s", input);
        
        prods[i].lhs = input[0];
        prods[i].count = 0;
        
        int len = strlen(input);
        int idx = 0;
        for(j = 3; j <= len; j++) {
            if(input[j] == '|' || input[j] == '\0') {
                prods[i].rhs[prods[i].count][idx] = '\0';
                prods[i].count++;
                idx = 0;
            } else {
                prods[i].rhs[prods[i].count][idx++] = input[j];
            }
        }
    }
    
    printf("\nOriginal CFG:\n");
    for(i = 0; i < n; i++) {
        printf("%c -> ", prods[i].lhs);
        for(j = 0; j < prods[i].count; j++) {
            printf("%s", prods[i].rhs[j]);
            if(j < prods[i].count - 1) printf(" | ");
        }
        printf("\n");
    }
    
    printf("\nStep 1: Remove epsilon productions\n");
    printf("Step 2: Remove unit productions\n");
    for(i = 0; i < n; i++) {
        for(j = 0; j < prods[i].count; j++) {
            if(strlen(prods[i].rhs[j]) == 1 && prods[i].rhs[j][0] >= 'A' && prods[i].rhs[j][0] <= 'Z') {
                printf("  Unit production found: %c -> %s\n", prods[i].lhs, prods[i].rhs[j]);
            }
        }
    }
    
    printf("\nConverting to CNF (Chomsky Normal Form):\n");
    printf("CNF Rules: A -> BC  or  A -> a\n\n");
    
    printf("CNF Productions:\n");
    for(i = 0; i < n; i++) {
        for(j = 0; j < prods[i].count; j++) {
            char* rhs = prods[i].rhs[j];
            int len = strlen(rhs);
            
            if(len == 1) {
                if(rhs[0] >= 'a' && rhs[0] <= 'z') {
                    printf("%c -> %s  (Valid CNF)\n", prods[i].lhs, rhs);
                }
            } else if(len == 2) {
                if(rhs[0] >= 'A' && rhs[0] <= 'Z' && rhs[1] >= 'A' && rhs[1] <= 'Z') {
                    printf("%c -> %s  (Valid CNF)\n", prods[i].lhs, rhs);
                }
            } else {
                printf("%c -> %s  (Needs conversion)\n", prods[i].lhs, rhs);
                
                char prev = prods[i].lhs;
                for(k = 0; k < len - 1; k++) {
                    char newVarName = newVar++;
                    if(k == len - 2) {
                        printf("  %c -> %c%c\n", prev, rhs[k], rhs[k+1]);
                    } else {
                        printf("  %c -> %c%c\n", prev, rhs[k], newVarName);
                        prev = newVarName;
                    }
                }
            }
        }
    }
    
    printf("\nCNF Conversion Complete!\n");
    return 0;
}
```

## Conclusion
The CFG to CNF conversion is a fundamental transformation in compiler design:

1. **Simplification**: Removes unnecessary complexity from grammars
2. **Parsing**: Enables efficient parsing algorithms like CYK
3. **Standardization**: Provides a uniform representation
4. **Applications**: Used in natural language processing and compiler construction

CNF ensures that parsing can be done efficiently with time complexity O(n³) using dynamic programming.
